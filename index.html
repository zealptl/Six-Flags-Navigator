How to use Six Flags Navigator?


*Currently, this app only supports paths between max thrill rides, but additional features are coming*


  1. To find the shortest path between to rides, enter the ride numbers separated by a coma.
  2. To find the shortest that start at a certain ride and visits all other max thrill rides, enter the ride number you want to start at.
